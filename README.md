# Customer Churn Analysis Dashboard using Power BI

## Overview

This project presents an interactive Customer Churn Analysis Dashboard developed using Power BI. The dashboard helps identify customer retention patterns, analyze churn behavior, and generate business recommendations for reducing customer attrition.

The dashboard enables organizations to understand factors influencing churn and make data-driven decisions to improve customer satisfaction and retention.

---

## Objectives

* Analyze customer churn patterns.
* Identify high-risk customer segments.
* Understand factors influencing customer retention.
* Generate business recommendations for churn reduction.
* Build interactive visualizations for business decision-making.

---

---

## Key Insights

* Month-to-month customers show the highest churn rates.
* New customers are more likely to churn.
* Customers using electronic checks have higher churn.
* Higher monthly charges increase churn probability.
* Lack of technical support contributes to customer attrition.
* Fiber optic customers exhibit higher churn rates.

---

## Business Recommendations

* Promote long-term contracts.
* Improve customer onboarding.
* Optimize pricing strategies.
* Strengthen technical support.
* Target high-risk customer groups.
* Improve service quality.
